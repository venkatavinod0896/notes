HTTP status codes
==================
1XX -> Informational
2XX -> Success
200 -> OK
201 -> Created
204 -> No content
3XX -> Redirectional
304 -> Not modified, take it from cache
301 -> Moved permanently
4XX -> Client side error
401 -> Unauthorised, please provide your authentication(Who are you?)
403 -> Forbidden, I know who are you? but I am not allowing you
404 -> Not found
405 -> Method not allowed
5XX -> Server side error
500 -> Internal server error
502 -> Bad gateway
504 -> Gateway timeout
503 -> Service unavailable
