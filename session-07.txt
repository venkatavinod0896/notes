custom systemctl service
=======================
[Unit]
Description=Expense Backend Service

[Service]

====================

forward proxy
=============
client(Mobile) -> server(Prime)
VPN

1. cleint aware
2. hiding client identity
3. changing geo location
4. traffic monitoring
5. content restrictions, filtering, etc..

reverse proxy
===============
1. server aware
2. load balancing
3. SSL/TLS/HTTPS termination
4. HTTP server
5. cache
